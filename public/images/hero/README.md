# Hero Images

Place your hero/banner images here:

- `hero-main.jpg` - Main homepage hero image
- `hero-about.jpg` - About page hero
- `hero-contact.jpg` - Contact page hero
- `hero-mobile.jpg` - Mobile-optimized hero image

Recommended specs:
- Desktop: 1920x1080px or larger
- Mobile: 800x600px
- Format: JPG (optimized for web)
- Max file size: 500KB per image
