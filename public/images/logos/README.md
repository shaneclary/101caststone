# Logos

Place your company logos here:

- `logo.svg` or `logo.png` - Main company logo
- `logo-white.svg` or `logo-white.png` - White version for dark backgrounds
- `logo-icon.svg` or `logo-icon.png` - Icon/favicon version
- `logo-horizontal.svg` - Horizontal lockup
- `logo-vertical.svg` - Vertical lockup

Recommended formats: SVG for scalability, PNG with transparency as backup
