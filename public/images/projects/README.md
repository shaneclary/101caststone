# Featured Projects

Place your completed project photos here:

Name format: `project-name-01.jpg`, `project-name-02.jpg`, etc.

Examples:
- `mediterranean-estate-01.jpg`
- `luxury-villa-exterior-01.jpg`
- `historic-restoration-01.jpg`

Include:
- Exterior shots showing installed products
- Detail shots of columns, cornices, etc.
- Before/after if available
- Wide shots showing full context

Recommended specs:
- Size: 1600x1200px or larger
- Format: JPG
- High quality, professional photography
